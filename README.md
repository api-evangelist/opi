# Open Programmable Infrastructure (opi)
The Open Programmable Infrastructure (OPI) project is a Linux Foundation initiative demonstrating real-world use cases for offloading key infrastructure management services from host CPUs to DPUs and IPUs. It enables hardware-accelerated security, load balancing, and API gateway functions.
